# AI-Driven Dynamic Pricing Engine

This project focuses on building an advanced dynamic pricing system
using Machine Learning and Reinforcement Learning to optimize product
prices under uncertain and changing demand.

## Problem Statement
Fixed pricing strategies fail to adapt to demand fluctuations,
inventory constraints, and customer behavior, leading to revenue
and profit loss.
